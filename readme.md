## 使用教程
本项目包含下列文件夹：
- Check-Excel
- Datasets
- deeplearning_model
- img-test
- src
### Check-Excel
保存签到系统自动生成的excel签到名单
### Datasets
用于保存签到系统的人脸数据库
### deeplearning_model
用于保存系统所使用的的深度学习模型
### img-test
用于保存系统测试的案列
### src
用于保存代码 
运行以下代码文件，既可以使用本系统： 
- camer_demo.py :运行系统
- IP_cam_demo.py :测试手机端Camon live Stream与电脑的图传
- DataSetPreProcess.py :数据集预处理脚本
- 其他文件为辅助文件
